# Avogadro-s-number
This project estimates **Avogadro's number** using microscopic images and image processing techniques, implemented in **Java**.

## Project Overview
The program analyzes microscopic images of particles and extracts statistical data to estimate Avogadro's number.

## Technologies
- Java
- Image Processing

## How to Use
Upload microscopy images and run the main Java class.

## Notes
This is an experimental and educational project.
